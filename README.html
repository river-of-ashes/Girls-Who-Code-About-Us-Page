Girls Who Code

This website was made as a part of Girls Who Code at the South Brunswick Public Library as an effort to bring awareness to make the world a better place

Girls Who Code is an organization founded with the goal to close the gender gap in technology. They encourage female engineers in the United States of America. More information about Girls Who Code can be found here.
